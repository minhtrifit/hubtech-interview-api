## 📦 Project setup <a name="project-setup"></a>

⚙️ Config [.env]() file in client dir with path `/.env`:

```bash
CLIENT_URL=http://localhost:3000
PORT=5500
DB_HOST=localhost
DB_PORT=5432
DB_USERNAME=admin
DB_PASSWORD=123456
DB_NAME=hubtech_interview
DB_SYNCHRONIZE=true
```

📥 Installation packages:

Intall packages

```console
npm install
```

Run API project

```console
npm run start
```
